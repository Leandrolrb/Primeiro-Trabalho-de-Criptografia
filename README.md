## Descrição

Este repositório contém o **Primeiro Trabalho de Criptografia**, desenvolvido como atividade da disciplina **Segurança de Sistemas de Computação**.

O objetivo deste trabalho é aplicar conceitos relacionados à criptografia, segurança da informação e proteção de dados, utilizando os conhecimentos adquiridos ao longo da disciplina.

## Objetivos

* Compreender os princípios da criptografia.
* Implementar algoritmos ou técnicas criptográficas.
* Analisar mecanismos de segurança.
* Aplicar boas práticas de proteção de informações.
